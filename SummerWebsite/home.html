Programming Projects 2017-2018

I have worked on many programming projects over the course of my Junior year. These are a couple ones that I think are the most interesting:

  • Recursive Tic Tac Toe: A mobile app game based on an interesting variant of the classic game of tic tac toe.
  • Dragon Curve Generator: A python program that generates various tiers of the mathematical curve known as the dragon curve.
  • Mondrian Art Generator: A python program that allows you to draw Mondrian Art
  • Snake Genetic Algorithm: A python program that uses a genetic algorithm along with a neural network to try and create a snake that can play the game snake.
